Projeto Casa

SOBRE O PROJETO

Meu primeiro projeto no AutoCad que fiz durante meu estágio na JCGontijo. O projeto retrata a parte estrutural de uma casa e a fundação profunda dela.

ESTRUTURA

Projeto de estrutura da casa fiz conforme as medidas do desenho, utilizei a nomeclatura de medida de centimetros pois ficou visualmente melhor, adicionei as cotas somente no projeto de fundação para melhor visualização.

Fundação Profunda

As fundações profundas utilizando um circulo em layer verde para representa-las, vigas baldrame representadas por layer amarelos.

Cotas

Cotas personalizadas utilizando o comando "DLI", fiz a ordem alfabetica e a numerica para melhor indentificação, representadas por layers vermelhos, as cotas seguiram as medidas que para fazer o projeto foram de grande ajuda.

Layers

Formatação dos layers que utilizei no projeto para separar cada parte da fundação, as cores foram das mais fracas as mais fortes para ficar melhor de se indentificar com os plot do AutoCad.
